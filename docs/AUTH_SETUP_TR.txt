TicarNet Online - Kimlik dogrulama kurulumu (Firebase Auth)

1) Onerilen yapi
- Kayit / giris / sifremi unuttum: Firebase Authentication
- Destek talepleri e-postasi: SMTP (opsiyonel, yoksa ticket kuyrukta kalir)

2) Zorunlu ortam degiskenleri (auth icin)
- FIREBASE_AUTH_ENABLED=true
- FIREBASE_WEB_API_KEY=<Firebase Web API Key>
- CLIENT_URL=https://alanadiniz
- RESET_LINK_BASE_URL=https://alanadiniz
- CORS_ALLOWED_ORIGINS=https://alanadiniz
- JWT_SECRET=<en az 32 karakter>
- HEALTHCHECK_TOKEN=<en az 16 karakter>

3) Opsiyonel SMTP (destek e-postasi icin)
- SMTP_USER=mustafaard76@gmail.com
- SMTP_APP_PASSWORD=<gmail app password>
- MAIL_FROM="TicarNet Online <mustafaard76@gmail.com>"
- SUPPORT_INBOX_EMAIL=mustafaard76@gmail.com

Not: SMTP kapaliysa auth akislarinda sorun olmaz, ancak destek ticket e-postasi otomatik gitmez.

4) Sifremi unuttum akisi
- Oyuncu e-postayi girer.
- Backend Firebase'e PASSWORD_RESET istegi yollar.
- Firebase kullaniciya e-posta gonderir.
- Kullanici linkten donunce frontend tokeni backend'e verir.
- Backend Firebase uzerinden tokeni dogrular ve yeni sifreyi kaydeder.

5) E-posta kurali
- Tum gecerli e-posta domainleri desteklenir.
