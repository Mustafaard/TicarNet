TicarNet Online - Kimlik doğrulama ve şifre sıfırlama kurulumu

1) Görsel dosyaları
- Auth arka planı: public/auth/auth-background.png
- Splash logosu: public/splash/logo.png
- Splash arka planı: public/splash/background.jpg

2) Sunucu ayarı
- server/.env.example dosyasını server/.env olarak kopyalayın.
- Aşağıdaki alanları doldurun:
  - SMTP_USER=mustafaard76@gmail.com
  - SMTP_PASS=Gmail App Password (16 haneli)
  - MAIL_FROM="TicarNet Online <mustafaard76@gmail.com>"
  - JWT_SECRET=uzun-gizli-anahtar
  - JWT_EXPIRES=30d
  - ADMIN_EMAILS=mustafaard76@gmail.com

3) Çalıştırma
- Tüm sistemi birlikte çalıştırın:
  npm run dev:all
- Yalnızca API için:
  npm run dev:api

4) Şifremi unuttum akışı
- Oyuncu e-posta adresini girer.
- Backend, SMTP ile şifre sıfırlama bağlantısı gönderir.
- Bağlantı adresi, talebin geldiği cihaz/origin adresine göre üretilir.
- E-posta bağlantısı uygulamayı token ile açar.
- Oyuncu yeni şifresini belirler ve backend token doğrulamasını tamamlayarak şifreyi günceller.

5) Güvenlik kuralları
- Yalnızca Gmail adresi kabul edilir.
- Aynı cihaz, yerel ağ (Wi-Fi alt ağı), istemci public IP ve sunucunun gördüğü IP kapsamında en fazla 2 hesap oluşturulabilir.
- Şifreler backend tarafında hashlenir.
- Sıfırlama token'ları tek kullanımlıktır ve süre sınırlıdır.
